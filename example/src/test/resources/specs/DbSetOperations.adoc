= DbUnit operations

https://dbunit.sourceforge.net/dbunit/components.html#databaseoperation[DbUnit operations] allows to append, update and delete rows.

[#before]
.Before each example
****
Given:

.product
[{db-set}]
,===
id, name, price

1, Adam, 10
2,  Bob, 20
,===
****

== Append rows
[source,asciidoc]
--
include::DbSetOperations.adoc[tags=db-insert]
--

.Append rows
====
When append row:

//tag::db-insert[]
.product
[e-db-set=, e-operation=insert]
,===
id, name, price

3, Carl, 30
,===
//end::db-insert[]

Then row added:

.product
[e-db-check=]
,===
id, name, price

1, Adam, 10
2,  Bob, 20
3, Carl, 30
,===
====

== Update rows
[source,asciidoc]
--
include::DbSetOperations.adoc[tags=db-update]
--

.Update rows
====
When update row

//tag::db-update[]
.product
[e-db-set=, e-operation=update]
,===
id, name, price

2, Bob2, 22
,===
//end::db-update[]

Then second row updated

.product
[e-db-check=]
,===
id, name, price

1, Adam, 10
2,  Bob2, 22
,===
====

== Delete rows
[source,asciidoc]
--
include::DbSetOperations.adoc[tags=db-delete]

.Delete rows
====
When delete row

//tag::db-delete[]
.product

2
,===
